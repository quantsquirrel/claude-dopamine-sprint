# Quiz: 5분 퀵 퀴즈

## Overview

학습한 토픽에서 랜덤으로 5문제를 출제하는 퀵 퀴즈 모드입니다.
30분 풀 스프린트가 부담스러울 때 퀴즈만 빠르게 풀 수 있습니다.

---

## Prerequisites (매 실행 시 반드시 수행)

1. **state.json 로드**: Read `~/.claude/claude-dopamine-sprint/state.json`
   - 파일이 없으면 안내: "아직 학습 기록이 없어요! `/sprint`로 첫 학습을 시작해보세요."
   - 완료된 토픽이 하나도 없으면 안내: "완료된 토픽이 없어요! 먼저 `/sprint`로 학습을 진행해주세요."

2. **curriculum.json 로드**: Read `${PLUGIN_ROOT}/data/curriculum.json`
   - PLUGIN_ROOT = 이 SKILL.md가 위치한 플러그인의 루트 디렉토리. 이 파일 기준으로 `../../data/curriculum.json`

3. **extensions 로드**: Glob `${PLUGIN_ROOT}/data/extensions/*.json` → 있으면 각각 Read

## 토픽 및 퀴즈 선택

1. **$ARGUMENTS 파싱**:
   - 토픽 ID가 지정된 경우: 해당 토픽의 quiz 배열에서 출제
   - `"all"`: 모든 토픽(완료 여부 무관)의 quiz에서 랜덤 출제
   - 인수 없음: 완료된 토픽(`progress`에서 `status: "completed"`)의 quiz에서 랜덤 출제

2. **스페이스드 리피티션 (간격 반복)**:
   - state.json의 `history`에서 각 토픽별 마지막 학습일(`date`)과 퀴즈 정답률을 계산한다.
   - 우선순위 점수 = `(오늘 - 마지막학습일) × (1 - 정답률)` (오래되고 틀린 것이 우선)
   - 점수가 높은 토픽의 퀴즈를 우선 출제 대상으로 선택한다.

3. **약점 집중 모드**:
   - 정답률 50% 이하인 토픽의 퀴즈를 추가 우선 출제한다.
   - 약점 토픽 + 간격 반복 우선순위로 상위 퀴즈를 선택하고, 나머지를 랜덤으로 채워 총 5문제를 구성한다.
   - 약점 토픽이 없으면 간격 반복 우선순위 → 랜덤 순서로 구성한다.

4. **5문제 구성**:
   - 대상 토픽들의 quiz 배열을 모두 합친 뒤, 위 우선순위에 따라 5문제를 선택한다.
   - 대상 퀴즈가 5개 미만이면 있는 만큼만 출제한다.

## 퀴즈 진행

각 문제(1~5번)에 대해:

1. 문제 번호와 함께 quiz의 **question**을 표시한다.
   ```
   📝 문제 {n}/5
   Q: {question}
   ```

2. AskUserQuestion으로 출제:
   - **options**: 선택지 3개 제공: 정답 1개 + quiz의 `distractors` 배열에서 오답 2개를 사용한다. `distractors`가 없으면 즉석에서 그럴듯한 오답 2개를 생성한다.
   - **중요**: 정답 위치를 랜덤하게 배치한다 (항상 첫 번째가 정답이면 안 됨)

3. 즉시 정답/오답 피드백:
   - **정답**: "정답이에요!" + 간단한 해설. 이전에도 맞춘 적 있으면 "연속 정답! 마스터 수준이에요!" 추가 피드백.
   - **오답**: "아쉬워요! 정답은 {correct_answer}에요." + 해설 + "왜 이 답이 맞는지" 한 줄 설명 추가.

4. **적응형 난이도**:
   - history에서 해당 토픽의 이전 정답률이 100%이면, 퀴즈 후 즉석 보너스 문제를 1개 추가 출제한다.
   - 보너스 문제는 curriculum에 없는 심화 문제를 해당 토픽의 key_points와 doc_url 기반으로 즉석 생성한다.
   - 보너스 문제는 state.json에 기록하지 않는다 (순수 도전 목적).

## 최종 결과

모든 문제 완료 후 결과를 표시한다:

```
═══════════════════════════════
📝 퀵 퀴즈 완료!
✅ 5문제 중 {맞춘수}개 정답! ({정답률}%)
═══════════════════════════════
```

정답률에 따른 피드백:
- **100%**: "완벽해요! 마스터 수준이에요!"
- **80% 이상**: "훌륭해요! 거의 다 맞췄어요!"
- **60% 이상**: "좋아요! 조금만 더 복습하면 완벽해질 거예요."
- **60% 미만**: "복습이 필요해요! `/sprint`로 다시 학습해보는 건 어떨까요?"

## state.json 업데이트

퀴즈 완료 후 아래 항목을 업데이트하고 Write로 `~/.claude/claude-dopamine-sprint/state.json`에 저장한다.
쓰기 전 `state.json.bak`으로 백업한다 (Bash `cp`). 쓰기 성공 후 백업을 삭제한다:

- **history에 추가**:
  ```json
  {"date": "YYYY-MM-DD", "topic": "quiz-mixed", "duration": 5, "quizScore": 맞춘수, "quizTotal": 전체문제수, "type": "quick-quiz"}
  ```
  (특정 토픽 지정 시 `"topic"` 값은 해당 토픽 ID)

- **totalStudyMinutes += 5**

- **totalSessions += 1**

- **스트릭은 업데이트하지 않음**: 퀵 퀴즈는 스트릭에 포함되지 않는다. 풀 스프린트(`/sprint`)만 스트릭 대상이다.

- **tils는 추가하지 않음**: 퀵 퀴즈에서는 TIL을 기록하지 않는다.

## Important Notes

- **모든 출력은 한국어**로 한다.
- **AskUserQuestion은 반드시 사용**한다 — 사용자 인터랙션이 핵심이다.
- **퀴즈 선택지**를 만들 때 정답 위치를 랜덤하게 배치한다 (항상 첫 번째가 정답이면 안 됨).
- **스트릭은 업데이트하지 않는다** — 풀 스프린트만 스트릭 대상이다.
- **약점 집중**: 이전에 틀린 적 있는 퀴즈를 우선 출제하여 복습 효과를 극대화한다.
